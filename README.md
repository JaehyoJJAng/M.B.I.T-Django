# 세팅

### python3 및 가상환경 설치  
```shell
# 패키지 업데이트 & 파이썬3 설치
$ yum update -y && yum install -y python3

# 가상환경 생성
$ python3 -m venv venv

# 가상환경 접속
$ source ./venv/bin/activate

# 라이브러리 설치
$ pip install -r requirements.txt
```
